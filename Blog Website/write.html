Instructions on How to use Writing tools

  1. 1. You can still edit text in many elements you added.
  2. 2. The first form is for inserting the blog cover image.
  3. 3. The second form is used for inserting image, paragraph, heading etc.
  4. 4. Quickly clicks 3 times to delete the elements added by second form
  5. 5. In writing you can use <b> and </b> for bold content in paragraph.
  6. 6. In inserting Image, we want 3 parameters : url;;;width(in px);;;height(in px) [In this format only]
  7. 7. For inserting list, you can break to new points with ;;; and note that list is not editable after insert
  8. 8. Clear Button is to clear the textbox and Submit button is to finish editing and get download option of your blog.
